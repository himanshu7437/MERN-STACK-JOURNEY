# HTML Elements: Block and Inline

This document provides a quick reference for commonly used HTML block and inline elements, categorized by their purpose and functionality.

---

## Block Elements (Most Commonly Used First)

| Element       | Description                                                                 |
|---------------|-----------------------------------------------------------------------------|
| `<div>`       | A generic container for flow content.                                      |
| `<p>`         | Paragraph.                                                                |
| `<h1>` - `<h6>` | Headings (from most important to least important).                        |
| `<ul>`        | Unordered list.                                                           |
| `<ol>`        | Ordered list.                                                             |
| `<li>`        | List item.                                                                |
| `<form>`      | A section containing form controls.                                       |
| `<table>`     | Table.                                                                    |
| `<section>`   | A standalone section of a document.                                       |
| `<header>`    | Container for introductory content or navigation links.                   |
| `<footer>`    | Footer of a section or page.                                              |
| `<nav>`       | Section of navigation links.                                              |
| `<article>`   | A self-contained composition in a document.                               |
| `<aside>`     | Information indirectly related to the main content.                       |
| `<main>`      | The main content of a document.                                           |
| `<fieldset>`  | Groups form controls under a common name.                                 |
| `<blockquote>`| A block of text that is a quotation from another source.                  |
| `<pre>`       | Preformatted text.                                                       |
| `<canvas>`    | Container used to draw graphics via JavaScript.                           |
| `<dl>`        | Description list.                                                        |
| `<dt>`        | Term in a description list.                                              |
| `<dd>`        | Description in a description list.                                       |
| `<figure>`    | Content referenced from the main content.                                 |
| `<figcaption>`| Caption for a `<figure>` element.                                         |
| `<address>`   | Contact information for the document author or owner.                    |
| `<hr>`        | Thematic break or horizontal rule.                                        |
| `<tfoot>`     | Footer of a table.                                                       |

---

## Inline Elements (Most Commonly Used First)

| Element       | Description                                                                 |
|---------------|-----------------------------------------------------------------------------|
| `<a>`         | Anchor or hyperlink.                                                      |
| `<img>`       | Image.                                                                    |
| `<span>`      | Generic inline container.                                                 |
| `<input>`     | Input field.                                                              |
| `<label>`     | Label for a form element.                                                 |
| `<strong>`    | Strong emphasis.                                                         |
| `<em>`        | Emphasized text.                                                         |
| `<br>`        | Line break.                                                              |
| `<code>`      | Code snippet.                                                            |
| `<b>`         | Bold text.                                                               |
| `<i>`         | Italic text.                                                             |
| `<u>`         | Underlined text.                                                         |
| `<small>`     | Smaller text.                                                            |
| `<sub>`       | Subscript.                                                               |
| `<sup>`       | Superscript.                                                             |
| `<mark>`      | Marked or highlighted text.                                              |
| `<q>`         | Short inline quotation.                                                  |
| `<cite>`      | Citation.                                                                |
| `<kbd>`       | Keyboard input.                                                          |
| `<samp>`      | Sample output.                                                           |
| `<var>`       | Variable in a mathematical expression or programming context.            |
| `<time>`      | Time.                                                                    |
| `<abbr>`      | Abbreviation.                                                           |
| `<data>`      | Machine-readable translation of content.                                 |
| `<acronym>`   | Acronym (Not supported in HTML5).                                        |

---

## Notes:
- Some elements like `<acronym>` are deprecated in HTML5 and should be avoided in modern web development.
- For additional styling or detailed examples, consider creating a live demo using CodePen or including inline examples.
